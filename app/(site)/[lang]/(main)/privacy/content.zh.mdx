## 本网站的用途

本网站是 Keezaa 的产品介绍页面，提供有关应用的信息、App Store 下载链接以及通过电子邮件联系的方式。

## 可能处理的数据

当你访问本网站时，作为托管服务提供商的 GitHub Pages 会处理技术上必需的连接数据，以便正确交付页面。根据 GitHub 的说明，其中也包括出于安全目的可能被记录和保存的 IP 地址。此外，还可能处理访问时间、访问内容以及浏览器和设备信息。

如果你通过电子邮件与我们联系，我们还会处理你发送的信息，以便回复你的请求。

## TelemetryDeck

本网站使用 TelemetryDeck 进行注重隐私的访问量统计。

根据 TelemetryDeck 的说明，不会存储 IP 地址，也不会使用 Cookie。其唯一目的是帮助我们更好地理解并改进本网站。

更多信息：

- [TelemetryDeck Privacy FAQ](https://telemetrydeck.com/docs/guides/privacy-faq/)
- [TelemetryDeck Privacy Policy](https://telemetrydeck.com/privacy/)

## 第三方服务

### Google Fonts / Material Symbols

本网站会从 Google 加载字体和图标字体，以确保页面可以正确显示。因此，你的浏览器可能会与 Google 的服务器建立连接。

更多信息：

- [https://policies.google.com/privacy](https://policies.google.com/privacy)
- [https://policies.google.com/privacy/frameworks](https://policies.google.com/privacy/frameworks)

## 无广告，无用户账户

本网站不使用用户账户，不使用广告追踪器，也不使用用于用户画像的 Cookie。

## 变更

如果网站内容或法律要求发生变化，本隐私政策可能会相应更新。
